What is this?

subleq.js is a simulator for the SUBLEQ language. The interpreter code is written in Javascript and runs entirely in the browser.

Usage

SUBLEQ

The language accepts a single command:

 SUBLEQ A,B,C

This subtracts the contents of memory address A from memory address B and stores the result in B. If this result is less than or equal to zero then branch to the location C.

Comments

Anything after a semicolon until a linebreak is a comment and ignored

; comments extend to the end of the line
Labels

You can use labels to make memory addressing easier

baz:               ; any word followed by a colon
SUBLEQ foo,bar,baz ; subtract foo from bar and jump to baz is LEQ zero
Data

Use the DATA statement to define a word in memory.

DATA 5      ; emits the value 5 at this location in memory
foo: DATA 6 ; you can also label the location of data elements
